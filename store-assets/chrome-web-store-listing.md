## Extension Name

Cookie Manager

## Short Description (132 chars max)

View, edit, search, import/export cookies. Profiles, real-time monitor, dark mode. Developer tool for cookie management.

## Detailed Description

Cookie Manager is a powerful yet simple Chrome extension for comprehensive cookie management. Built for developers, testers, and power users who need full control over browser cookies.

**Key Features:**

🍪 **View & Search Cookies**

- View all cookies for the current page or across all domains
- Instant search by name, value, or domain
- Filter by domain with searchable dropdown
- See Secure (S) and HttpOnly (H) flags at a glance

📝 **Edit & Create Cookies**

- Edit any cookie field: name, value, domain, path, expiration
- Create new cookies with full attribute control
- Set Secure, HttpOnly, and SameSite attributes
- Toggle between session and persistent cookies

📋 **Copy & Export**

- One-click copy cookie value, name, or full cookie string
- Export in JSON, Netscape/Mozilla, or HTTP Header format
- Download exports as files or copy to clipboard

📥 **Import Cookies**

- Import from JSON, Netscape, or Header format
- Paste text or upload files
- Auto-detect format

💾 **Cookie Profiles**

- Save current cookies as named profiles
- Restore profiles to switch between cookie sets
- Perfect for testing different user sessions

🔀 **Clone Cookies**

- Clone any cookie to a different domain with one click
- Pick target from open tabs or enter URL manually
- Auto-derives domain and strips Secure flag for HTTP targets
- Live domain preview as you type

🔒 **Incognito Support**

- Full cookie isolation between normal and incognito windows
- Separate cookie store per context — no cross-contamination
- All views (popup, side panel, full page) respect incognito boundary

📡 **Real-time Monitor**

- Watch cookie changes as they happen
- See which cookies are set, deleted, or expired
- Pause/resume monitoring

🎨 **Multiple Views**

- Popup: Quick access from toolbar
- Side Panel: Persistent view alongside your browsing
- Full Page: Complete management dashboard with all features

🌍 **Bilingual**: English and Vietnamese
🌙 **Dark Mode**: Light, dark, or follow system theme

**Privacy First:**

- Zero data collection — all data stays on your device
- No analytics, no tracking, no remote servers
- Open source — verify our claims yourself

**Permissions Explained:**

- "cookies": Read and manage browser cookies (core feature)
- "storage": Save your preferences locally
- "tabs": Detect current tab to show relevant cookies
- "host_permissions": Required by Chrome Cookies API for cross-domain access
